Trigonometría

Teorema de Pitágoras:
a² + b² = c²

Funciones Trigonométricas:
• sen = opuesto / hipotenusa
• cos = adyacente / hipotenusa
• tan = opuesto / adyacente

Ley de Senos:
a/sen A = b/sen B = c/sen C

Ley de Cosenos:
c² = a² + b² - 2ab cos C

Identidades Trigonométricas:
Recíprocas:
• csc = 1/sen
• sec = 1/cos
• cot = 1/tan

Cociente
• tan = sen/cos

Pitagóricas:
• sen² + cos² = 1
